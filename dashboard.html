Auto-Repair Strategies

🔄
META_API_DEPRECATION
Detect deprecated fields → remap to v19 equivalents → migrate endpoint → test run 100 records → full reprocess
API_VERSION_DEPRECATED · v16 · deprecated
🔑
TOKEN_EXPIRED
Cannot auto-repair — opens Slack OAuth modal, sends user a re-auth link, resumes pipelines on completion
OAUTH_TOKEN_EXPIRED · UNAUTHORIZED · token
⏱️
RATE_LIMIT
30s exponential backoff → reprocess only the failed time window
RATE_LIMIT_EXCEEDED · rate limit
🛠️
GENERIC
Test run with 100 records → if passes, trigger full reprocess; if fails, flag for manual review
FIELD_NOT_FOUND · all other codes

Health Score Formula

score = 100
    − failed_pipelines × 10
    − warning_pipelines × 4
    − expiring_tokens × 3
// floor: 0
